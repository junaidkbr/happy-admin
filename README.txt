Happy Admin simplify your WordPress admin by removing unnecessary clutter and dimming high contrast colors.

== Description ==

Clutters are distracting and high contrast colors will always steal your attention which usually results in lowered productivity and higher chances of distractions.
Happy admin removes the following things.
* Unnecessary banners or columns, this does not include the important WordPress columns.
* Reduces the contrast of attention stealing navigation items and notification badges.
* Disables admin toolbar if you pass in a certain URL variables. (Read `How To` section below)

== How To ==

`Happy Admin` automatically removes all the clutter and dim the high contrast elements, soon we will release a settings page where you may customize what you see and how your admin panel should look.

= Remove adminbar =

Sometimes you may want to remove the adminbar, it can be because you may want to check the layout or anything that's related to styling. You can do that by passing `?toolbar=off` to the URL and pressing enter. Your URL should look like this 'https://domain.example/?toolbar=off'.

== Installation ==

= Minimum requirements =

* PHP 7.2 or greater is recommended
* WordPress 5.2 is recommended

= Automatic Installation =

Automatic Installation is the easiest way to install the plugin. Signin to your WordPress dashboard, go to plugins and search for `Happy Admin.` Simply click on `install now` to install the plugin.
